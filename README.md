# 交大指南

##课题概述

本小组主要设计实现一个包含多个页面的校园生活网站，可能包括二手交易、校园情况、周边生活地点介绍、学习资料分享等多个板块。在网站设计和实现上将表现出小组成员对HTML标签的理解，以及运用CSS美化设计的意识，以及运用JavaScript增强人机交互的技术。本网站搭建主要是为了便于同学们更快适应校园生活。
